JSConTest - Test Framework - Basic

This test suite runs a lot of basic contracts against a set of values. It is expected to test 187 contracts. 29 should be verified, 147 should fail, and 11 should exit with an error. This is checked automatically, since two event handlers are registered. One checks if the contract behaves as expected. The other prints the usual contract output to the page.

If something unexpected happens, the first event handler will add an log message into the error box below. This may indicate, that JSConTest does not work correctly in your browser, either due to the fact that your browser does not support a feature JSConTest rely on, or because JSConTest has an error.
